# Fish Audio TTS Web

Лёгкая веб-морда на PHP для озвучки текста через Fish Audio API. https://fish.audio/ru/

Версия v7: добавлен обновлённый визуальный дизайн. Native multi-speaker режим Fish Audio сохранён.

## Что умеет

- Обычная озвучка одним голосом.
- Диалоги в человеческом формате: `М: текст` и `Ж: текст`.
- Native multi-speaker Fish Audio: один запрос, один цельный файл, без ffmpeg.
- Автоматический режим: сначала Fish native, если API вернул ошибку — резервная склейка реплик.
- Резервная склейка диалога в WAV без ffmpeg.
- Склейка/конвертация в MP3, если на сервере установлен `ffmpeg` и не запрещён `shell_exec`.
- Скачивание итогового файла и отдельных реплик в режиме склейки.
- Выбор модели: `s2.1-pro-free`, `s2.1-pro`, `s2-pro`, `s1`.
- Выпадающий список голосов: русские и английские пресеты.
- Возможность вставить свой `Voice ID / reference_id`.
- MP3/WAV/OPUS для обычной озвучки.
- MP3/WAV для диалогов.
- Скорость, громкость, temperature, top_p, latency.
- Кнопка подготовки английских слов: добавляет `[English]` перед латиницей.
- Панель кликабельных тегов: язык, speaker tags, интонации, смех, паузы, вдох/выдох и готовые примеры.
- Сохранение результата в папку `outputs/`.
- Простой пароль на вход, чтобы посторонние не тратили ваш API-ключ.
- Обновлённый адаптивный дизайн: карточки, градиентный фон, аккуратные кнопки и тёмная тема.


## Как работает native-диалог

В интерфейсе вы пишете так:

```text
М: Привет. Сегодня разберём слово [English]rainforest.
Ж: Дословно его иногда переводят как “дождевой лес”, но правильнее сказать: влажный тропический лес.
М: А [English]API лучше читать как эй-пи-ай [pause] а не АПИ.
Ж: Вот теперь звучит нормально. [laugh]
```

Сайт сам преобразует это в формат Fish Audio:

```text
<|speaker:0|>Привет. Сегодня разберём слово [English]rainforest.
<|speaker:1|>Дословно его иногда переводят как “дождевой лес”, но правильнее сказать: влажный тропический лес.
<|speaker:0|>А [English]API лучше читать как эй-пи-ай [pause] а не АПИ.
<|speaker:1|>Вот теперь звучит нормально. [laugh]
```

И отправляет `reference_id` как массив из двух голосов:

```json
"reference_id": ["VOICE_ID_МУЖЧИНЫ", "VOICE_ID_ЖЕНЩИНЫ"]
```

## Режимы диалога

### Авто: Fish native → склейка при ошибке

Рекомендуемый режим. Сначала пробует native multi-speaker. Если Fish Audio вернул ошибку, сайт автоматически пробует старый режим: генерирует каждую реплику отдельно и склеивает.

### Только Fish native multi-speaker

Один запрос к Fish Audio. Не нужен `ffmpeg`. Не создаются отдельные реплики. Если API вернул ошибку — сайт покажет ошибку без fallback.

### Только склейка реплик

Каждая реплика генерируется отдельным запросом, затем всё склеивается на сервере. WAV работает без `ffmpeg`. Для MP3 нужен `ffmpeg`.

## Установка

1. Загрузите папку на PHP-хостинг или VPS с PHP 8+ и расширением cURL.
2. Скопируйте конфиг:

```bash
cp config.example.php config.php
```

3. Откройте `config.php` и вставьте API key:

```php
'fish_api_key' => 'ВАШ_КЛЮЧ',
'app_password' => 'сложный-пароль',
```

4. Проверьте, что PHP может писать в папку `outputs/`:

```bash
chmod 775 outputs
```

5. Откройте `index.php` в браузере.

## Настройки

### API key

API-ключ хранится только в `config.php` на сервере и не отдаётся в браузер.

### Пароль

Если пароль не нужен, в `config.php` поставьте:

```php
'app_password' => '',
```

Но лучше оставить пароль, иначе любой посетитель страницы сможет тратить ваш Fish Audio API-ключ.

### Голоса

Список голосов можно менять в `config.php` в блоке `voice_presets`.

Для конкретного голоса Fish Audio используется `reference_id`.

## Английские слова в русском тексте

Чтобы модель не читала английское слово по буквам на русском, ставьте языковой тег:

```text
Сегодня разберём слово [English]rainforest.
А [English]API лучше читать как эй-пи-ай, а не АПИ.
```

В интерфейсе есть кнопка `Пометить английские слова [English]`, но после неё лучше проверить текст вручную: она может задеть бренды, ссылки и email.

## Теги эмоций и интонации

Примеры:

```text
[calm]
[confident]
[excited]
[curious]
[sarcastic]
[whisper]
[emphasis]
[laugh]
[sigh]
[gasp]
[inhale]
[exhale]
[pause]
```

Можно пробовать и более естественные описания:

```text
[whispers sweetly]
[laughing nervously]
[speaking slowly]
[teacher tone]
```

## MP3 для режима склейки

Для native multi-speaker MP3 приходит напрямую из Fish Audio.

Для резервной склейки MP3 нужен `ffmpeg`:

```bash
apt update
apt install -y ffmpeg
```

Если `ffmpeg` недоступен, сайт сохранит итоговый диалог в WAV.

## Файлы

- `index.php` — веб-интерфейс и логика.
- `shared_controls.php` — общие настройки генерации.
- `config.example.php` — пример конфига.
- `config.php` — ваш реальный конфиг, не публиковать.
- `outputs/` — сгенерированные аудиофайлы.
- `.htaccess` — закрывает `config.php` от прямого доступа на Apache.
