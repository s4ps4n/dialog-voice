# Fish Audio TTS Web v9

Лёгкая PHP-вебморда для Fish Audio API.

## Главное в v9

Интерфейс полностью разделён на два самостоятельных режима:

- **Диалог — два голоса**: сценарий `М:` / `Ж:`, отдельный мужской и женский голос, native multi-speaker и резервная склейка.
- **Один голос — дикторский текст**: отдельный редактор, один голос и свои заготовки.

Теги и готовые фразы больше не копируются «куда-то в буфер» и не относятся только к одному режиму. Кнопка вставляет текст **в редактор текущей вкладки и в позицию курсора**.

### Для диалога

Есть быстрые кнопки:

- `М: реплика`
- `Ж: реплика`
- `[calm]`
- `[pause]`
- `[English]`
- автоматическая пометка латиницы

Полная библиотека тегов и готовые диалоги находятся под редактором.

Пример:

```text
М: [calm] Привет. Сегодня разберём слово [English]rainforest.
Ж: [curious] А почему его нельзя переводить дословно?
```

### Для одного голоса

В отдельной вкладке доступны собственные теги и заготовки:

- новостной диктор;
- автоответчик;
- эмоциональные реплики;
- язык, паузы и эффекты.

## Mobile-first

- вкладки закрепляются сверху;
- основной редактор всегда расположен раньше настроек;
- быстрые теги прокручиваются горизонтально;
- большая кнопка генерации закреплена в нижней зоне большого пальца;
- на ПК редактор и настройки расположены в двух колонках;
- на ПК кнопка генерации находится в панели настроек;
- подсказки и примеры выводятся отдельными строками и больше не слипаются.

## Установка

1. Загрузите файлы на PHP-хостинг или VPS с PHP 8+ и расширением cURL.
2. Скопируйте:

```bash
cp config.example.php config.php
```

3. Вставьте ключ:

```php
'fish_api_key' => 'ВАШ_API_KEY',
'app_password' => 'пароль-на-вход',
```

4. Дайте PHP право записи в `outputs/`:

```bash
chmod 775 outputs
```

## Обновление с v8

Замените:

```text
index.php
shared_controls.php
README.md
```

`config.php` не заменяйте.
